\chapter*{Висновки}\label{ch:04}

У даній курсовій роботі було розгянуто проблему організації процесу розробки конструкторської документації. Дану проблему було запропоновано вирішувати за допомогою впровадження систем контролю версій, як спеціалізованих так і загальновживаних. В даній роботі було проведене обчислення кошторисів побудови системи організації з урахуванням мережевих, апаратних та програмних вимог для сучасних САПР, PDM та систем контролю версій. Два альтернативних варіанти підбору програмного забезпечення були реалізовані на основі платного ПЗ та з використанням безкоштовного. Ці підходи не можна вважати повнітю рівноцінними тому, що необхідно врахувати різну кінцефу функціональність отриманих систем та необхідну кваліфікацію ІТ відділу для організації та підтримання системи.

До важливих результатів даної роботи можна віднести:
\begin{enumerate}
 \item аналіз процесу розробки конструкторської документації
 \item огляд та порівняльний аналіз представлених на українському ринку систем для рішення відповідних задач супроводу процесу проектування та організації виробництва, таких як ЛОЦМАН:PLM та SolidWorks Enterprise PDM
 \item будову та застосування протоколу VPN, що призначений для організації вірутальних комп’ютерних мереж
 \item огляд необхідного та вибір оптимального апаратного забезпечення для клієнтської та серверної частини даної системи
 \item огляд та підбір безкоштовних аналогів для програмного забезпечення даної системи
 \item обчислення кошторисів розгортки системи для представлених варіантів на основі різного програмного забезпечення з наведеними оціночної вартості для кожного з них
 \item надання рекомендацій щодо доцільності використання поданих альтернативних варіантів
\end{enumerate}
