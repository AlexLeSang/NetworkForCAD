\chapter*{Висновки}\label{ch:04}

У цій курсовій роботі було розгянуто проблему організації процесу розробки конструкторської документації. Цю проблему було запропоновано вирішувати за допомогою впровадження систем контролю версій, як спеціалізованих так і загальновживаних. Також було проведене обчислення кошторисів побудови системи організації з урахуванням мережевих, апаратних та програмних вимог для сучасних САПР, PDM та систем контролю версій. Два альтернативних варіанти підбору програмного забезпечення були реалізовані на основі платного та безкоштовного ПЗ. Ці підходи не можна вважати повністю рівноцінними тому, що необхідно врахувати різну кінцеву функціональність отриманих систем та необхідну кваліфікацію ІТ відділу для розгортання та підтримання системи у належному робочому стані.

До важливих результатів нашої курсової роботи можна віднести:
\begin{enumerate}
 \item аналіз процесу розробки конструкторської документації
 \item огляд та порівняльний аналіз представлених на українському ринку систем для рішення відповідних задач супроводу процесу проектування та організації виробництва, таких як ЛОЦМАН:PLM та SolidWorks Enterprise PDM
 \item огляд будови та застосування протоколу VPN, що призначений для організації вірутальних комп’ютерних мереж
 \item огляд необхідного та вибір оптимального апаратного забезпечення для клієнтської та серверної частини даної системи
 \item огляд та підбір безкоштовних аналогів для програмного забезпечення даної системи
 \item обчислення кошторисів розгортки системи для представлених варіантів на основі платного та безкоштовного програмного забезпечення з наведенням оціночної вартості для кожного з представлених варіантів
 \item графічне відображення зміни вартості та очікуваної вигоди від застосування безкоштовного програмного забезпечення
 \item надання рекомендацій щодо доцільності використання поданих альтернативних варіантів
\end{enumerate}
